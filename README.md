# Fusion Tickets v1.0.3

Install the packages for the Bot
- `cd bot`
- `pnpm install --production`
- `Fill the env.`
- `pnpm run postinstall`

Install the packages for the portal

- `cd portal`
- `npm install`
- `npm run build`
- `npm link`

Go back to the bot dir
- `npm link @fusion-tickets/settings`
